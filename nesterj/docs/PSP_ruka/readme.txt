PSP版への移植はrukaが行いました。

オリジナルソースコードは NNNesterJ v0.23 です。

NNNesterJとの主な違いです
・オリジナルはC++で書かれています。PSP版は全てC言語に変更を行いました。
　(アドレス参照回数が減ってオリジナルより高速化されてるかも…)
・malloc(free)等のANSI関数が使われていません(一部自前で用意)。PSP版制限
　によりグローバル確保しています。
・バウンダリ違反コードを変更。

PROBLEM
・nes_apu.cのサンプリング周波数による処理の振り分けがテストで書いたコード
　のままになっている。
