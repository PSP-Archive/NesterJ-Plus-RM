--- supported images ---

  * NES ROM Image                       (*.NES, *.NEZ)
  * Disk System Image                   (*.NES, *.FDS, *.FAM)
  * VS-Unisystem ROM Image              (*.NES)
  * PlayChoice-10's splitted ROM Image  (*.PCT + *.BIN)
  * NES Audio Ripping Data              (*.NSF)

  fix header infomation if exist nestoy rom database 'nesdbase.dat'

--- supported devices ---

  * controller #1 and #2(with mic device)
  * insert coins in VS-Unisystem games

  * Arkanoid Paddle
  * Crazy Climber Controller
  * Datach Barcode Battler
  * Doremikko Keyboard
  * Exciting Boxing Controller
  * Family Basic Keyboard (with Data Recorder)
  * Family Trainer
  * Hyper Shot
  * Mahjong Controller
  * Oeka Kids Tablet
  * Optical Gun (Zapper)
  * Pokkun Moguraa Controller
  * Power Pad
  * Space Shadow Gun
  * Turbo File (with automatical bank switch)
  * VS Unisystem Zapper

--- extras ---

  * Game Genie
  * TCP/IP network play (LAN/Internet)
  * NSF Player
  * Movie Recorder/Player
  * ROM Database for "header correcting" and "external device selecting"

--- emulation core ---

  * Extra Sound (FDS, FME7, MMC5, N106, VRC6, VRC7) with NEZplug Sound Core
  * PAL emulation
  * Frame-IRQ (e.g. Dragon Quest 1/2)
  * DPCM-IRQ (e.g. Romancia)
  * VROM Bank Switch per drawing 8 dots (e.g. MOTHER)
  * Bank-Switched SaveRAM (e.g. Best Play - Pro Yakyuu Special)
  * Serial-Connected SaveRAM (e.g. Dragon Ball Z) (partial)
  * I/O Connected SaveRAM (e.g. Mindseeker, Kaijuu Monogatari)
